
# Projeto de Gerenciamento de Biblioteca

É comum que bibliotecas enfrentem desafios na organização de seu acervo, como controle de
empréstimos e eficiência no atendimento. Isso pode resultar em processos manuais demorados,
dificuldades na localização de materiais e extravios. Para solucionar o problema da escola, um
sistema de gerenciamento para a biblioteca automatizaria a administração do acervo, permitindo
buscas avançadas, reservas online, notificações automáticas e controle eficiente de empréstimos
e devoluções. Essa solução melhora a experiência dos leitores e otimiza o trabalho dos
bibliotecários, garantindo maior segurança dos dados, acessibilidade em dispositivos móveis e
desktops, e eficiência no gerenciamento do acervo e dos usuários.


## Rodando localmente

Clone o projeto

```bash
  git clone https://link-para-o-projeto
```

Entre no diretório do projeto

```bash
  cd biblioteca-backend
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm start
```
Abra outro terminal para rodar o front-end
```bash
  cd biblioteca-frontend
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```
